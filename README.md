## 系統簡介
這個專案提供松山機場國際線即時到站航班資訊，整合政府資料開放平台--松山機場國內線即時到站航班所提供的公開資料，讓使用者可以快速掌握航班動態、統計數據與最新快訊。

## 功能特色
- 即時航班資訊：查詢最新到站航班。
- 統計分析：準點率、延誤趨勢圖表。
## 技術架構
前端採用 **Next.js 14 + React**，並使用TailwindCSS 進行 UI 設計。 後端規劃使用**ASP.NET Core API** 搭配 **PostgreSQL** 資料庫。
